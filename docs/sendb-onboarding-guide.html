Sendb 시작 가이드 처음 사용자를 위한 단계별 안내 — 약 30분이면 모든 설정을 완료할 수 있습니다

Sendb는 고객 데이터 관리부터 이메일·알림톡 자동발송까지 하나의 화면에서 해결하는 영업 CRM입니다.
아래 순서대로 따라하면 서비스의 핵심 기능을 빠르게 경험할 수 있습니다.

가입→ 온보딩→ 레코드 등록→ 이메일 설정→ 첫 이메일 발송→ 자동화 연결→ 분석 확인
기본 설정
1
회원가입 & 온보딩
약 3분

/signup 에서 이름, 이메일, 비밀번호를 입력하고 가입합니다.

가입 즉시 온보딩 마법사가 시작됩니다 (5단계):

  1. 조직 정보 — 회사 이름, 업종, 규모를 입력합니다. 나중에 설정에서 수정 가능하니 대략 입력해도 됩니다.
  2. 워크스페이스 생성 — 첫 번째 워크스페이스 이름을 정합니다 (예: "영업관리", "마케팅"). 아이콘도 골라보세요.
  3. 속성(필드) 템플릿 — 업종별 추천 필드 세트를 선택합니다. 고객 관리에 필요한 필드(이름, 이메일, 회사, 직책 등)가 한번에 만들어집니다.
  4. 팀원 초대 — 함께 사용할 팀원 이메일을 입력합니다. 나중에 해도 됩니다.
  5. 완료 — 결과 확인 후 시작하기 클릭!
급하다면 상단의 "건너뛰기"로 온보딩을 생략할 수 있습니다. 설정은 언제든 워크스페이스 설정과 조직 설정에서 수정 가능합니다.
2
파티션 만들기 & 첫 레코드 등록
약 5분

/records (사이드바 "레코드") 페이지로 이동합니다.

파티션이란? 고객 데이터를 목적별로 나눈 공간입니다. 예를 들어:

  • "신규 리드" — 아직 연락하지 않은 잠재 고객
  • "미팅 진행중" — 상담 중인 고객
  • "계약 완료" — 성사된 거래

해볼 것:

  1. 좌측 패널에서 + 파티션 추가 클릭 → 이름 입력 → 생성
  2. 필요하면 + 폴더 추가로 파티션을 폴더로 묶을 수도 있습니다
  3. 상단 + 레코드 추가 클릭 → 고객 정보 입력 → 저장
한꺼번에 등록하려면? 상단 CSV 가져오기로 엑셀/CSV 파일을 업로드하세요. 필드 매핑을 확인하고 일괄 등록됩니다.
3
레코드 다뤄보기
약 3분

등록한 레코드로 기본 기능을 체험해봅니다:

  • 셀 클릭 → 인라인 편집 — 테이블에서 바로 값을 수정해보세요
  • 행 클릭 → 상세 보기 — 전체 필드, 메모, 발송이력을 한눈에 확인
  • 체크박스 선택 — 여러 레코드를 선택하면 상단에 일괄 삭제, 알림톡/이메일 발송 버튼이 나타납니다
  • 검색 & 필터 — 키워드 검색, 필드 조건 필터(같음, 포함, 비어있음 등)를 사용해보세요
  • 정렬 — 컬럼 헤더를 클릭하면 오름/내림차순 정렬됩니다
  • 뷰 모드 — 그룹화 가능한 필드(select 타입)가 있으면 그룹 뷰로 전환해보세요

4
이메일 API 연동
약 5분

Sendb는 NHN Cloud Email을 통해 이메일을 발송합니다. API 키를 등록해야 합니다.

  1. /email (사이드바 "이메일") 이동
  2. 우측 상단 톱니바퀴(⚙) 아이콘 클릭 → 설정 탭 진입
  3. NHN Cloud 콘솔에서 발급받은 Appkey와 Secret Key를 입력 → 저장
  4. 같은 설정 페이지에서 발신자 프로필을 등록합니다 (이름 + 이메일 주소)
NHN Cloud Email 서비스에 먼저 가입하고, 발신 도메인 인증을 완료해야 합니다. NHN Cloud 콘솔 → Notification → Email에서 진행합니다.
5
이메일 템플릿 만들기
약 5분
  1. /email → "템플릿" 탭 → + 새 템플릿
  2. 제목 입력 (예: "서비스 소개 안내")
  3. 비주얼 에디터에서 본문 작성. 일반 워드 편집처럼 텍스트를 입력하면 됩니다
  4. 수신자 맞춤 내용이 필요하면 #{이름}, #{회사} 같은 변수를 삽입합니다. 발송 시 레코드 데이터로 자동 치환됩니다
  5. 우측 미리보기로 결과를 확인합니다
  6. 발행 버튼으로 저장합니다
AI 활용 팁: 에디터 내 AI 패널에서 "B2B SaaS 소개 이메일 작성해줘"처럼 프롬프트를 입력하면 AI가 초안을 생성해줍니다. 조직 설정 → AI 탭에서 API 키 등록이 필요합니다.
6
첫 이메일 발송해보기
약 2분

두 가지 방법이 있습니다:

방법 A: 레코드 페이지에서 직접 발송

  1. /records에서 발송할 레코드를 체크박스로 선택
  2. 상단 이메일 발송 클릭
  3. 발신자 선택 → 템플릿 선택 → 수신 이메일 필드 지정 → 발송

방법 B: 레코드 상세에서 개별 발송

  1. 레코드 행 클릭 → 상세 다이얼로그에서 이메일 발송
발송 후 /email → "발송 이력" 탭에서 상태(대기/성공/실패), 읽음 여부, 클릭 수를 확인할 수 있습니다.
자동화 설정
7
이메일 자동발송 연결하기
약 5분

레코드가 생성/수정될 때 이메일이 자동으로 발송되도록 설정합니다.

  1. /email → "템플릿 자동발송" 탭 → + 연결 추가
  2. 연결명 입력 (예: "신규 리드 환영 이메일")
  3. 대상 파티션 선택 (예: "신규 리드")
  4. 사용할 이메일 템플릿 선택
  5. 트리거 선택:
    • 레코드 생성 시 — 새 고객이 등록되면 자동 발송
    • 레코드 수정 시 — 특정 필드가 변경되면 발송
    • 반복 발송 — 주기적으로 발송
  6. 수신 이메일 필드 지정 (어떤 필드에 수신자 이메일이 있는지)
  7. 필요하면 조건 추가 (예: 상태가 "신규"일 때만)
  8. 저장 → 활성화
클릭 추적: 연결 설정에서 "클릭 추적"을 ON으로 하면 이메일 내 링크 클릭 횟수를 추적할 수 있습니다.
후속 이메일: "후속 이메일 설정"을 켜면 수신자가 이메일을 읽은 후 N일 뒤에 자동으로 후속 이메일을 보낼 수 있습니다.
8
웹 폼으로 리드 자동 수집
약 5분

웹사이트에 폼을 달아서 방문자의 정보를 자동으로 레코드에 저장합니다.

  1. /web-forms (사이드바 "웹 폼") → + 새 폼 만들기
  2. 폼 이름 입력, 대상 파티션 선택
  3. 필드 추가: 이름, 이메일, 전화번호, 문의내용 등
  4. 각 폼 필드를 레코드 속성에 매핑 (폼의 "이메일" → 레코드의 "email" 필드)
  5. 저장 후 링크 버튼 클릭 → 공유 URL 또는 iframe 코드를 복사
  6. 홈페이지에 삽입하거나 URL을 공유
자동화 시너지: 웹 폼으로 레코드가 생성되면 → Step 7에서 설정한 자동발송이 트리거되어 환영 이메일이 자동 전송됩니다!

9
알림톡 연동 (선택)
약 10분

카카오 알림톡으로도 메시지를 보낼 수 있습니다.

  1. /alimtalk → "설정" 탭에서 NHN Cloud 알림톡 API 키 등록
  2. "발신프로필" 탭에서 카카오톡 채널 확인
  3. "템플릿" 탭 → + 템플릿 등록으로 메시지 템플릿 작성
  4. 템플릿은 카카오 검수를 통과해야 발송 가능합니다 (보통 1~2영업일)
  5. 승인된 템플릿으로 "연결 관리"에서 자동발송 설정 가능
10
AI 기능 활성화 (선택)
약 3분

AI 기능을 사용하면 이메일 작성, 제품 정보 생성, 기업 조사를 자동화할 수 있습니다.

  1. /settings/organization → "AI" 탭
  2. OpenAI 또는 Anthropic API 키 입력 → 저장
  3. 기본 AI 제공사 선택

활성화 후 사용 가능한 AI 기능:

  • AI 이메일 생성 — 템플릿 에디터에서 프롬프트로 이메일 초안 작성
  • AI 알림톡 생성 — 알림톡 템플릿 자동 생성
  • AI 자동발송 — 레코드 데이터 기반 맞춤 이메일 자동 생성+발송
  • AI 제품 정보 생성 — 제품 관리에서 웹 검색 기반 자동 생성
  • AI 기업 조사 — 레코드 상세에서 회사 정보 자동 수집
  • AI 웹 폼 필드 생성 — 프롬프트로 폼 필드 자동 구성
11
제품/서비스 등록 (선택)
약 3분

홍보할 제품이나 서비스를 등록하면 AI 이메일 작성 시 자동으로 참고합니다.

  1. /products (사이드바 "제품 관리") → + 제품 추가
  2. 제품명, 카테고리, 가격, 소개, 상세 설명 입력
  3. AI가 켜져 있으면 "AI 생성" 버튼으로 웹 검색 기반 자동 작성도 가능
12
대시보드에서 성과 확인
수시

발송이 진행되면 대시보드에서 성과를 확인합니다.

  • / 홈 대시보드 — 전체 요약 (레코드, 발송 현황, 최근 이력, 기간별 분석)
  • /email 이메일 대시보드 — 읽음률, 클릭률, 발송유형별 성과, 일별 추세
  • /logs 통합 발송 이력 — 알림톡+이메일 전체 이력을 한 곳에서 확인
핵심 지표: 읽음률(이메일을 열어본 비율)과 클릭률(읽은 이메일에서 링크를 클릭한 비율)을 주기적으로 확인하세요. 템플릿과 발송 타이밍을 조정하는 데 중요한 기준이 됩니다.

체크리스트

아래 항목을 순서대로 완료하면 Sendb를 충분히 활용할 수 있습니다.

  • 회원가입 완료
  • 온보딩 마법사 완료 (조직 + 워크스페이스 + 필드)
  • 파티션 1개 이상 생성
  • 레코드 5개 이상 등록 (수동 또는 CSV)
  • 이메일 API 키 등록
  • 발신자 프로필 등록
  • 이메일 템플릿 1개 생성
  • 첫 이메일 발송 성공
  • 자동발송 연결 1개 설정
  • 웹 폼 1개 생성 + 테스트 제출
  • (선택) AI API 키 등록
  • (선택) 알림톡 API 키 등록 + 템플릿 생성
  • (선택) 제품 1개 등록
  • (선택) 팀원 1명 초대
