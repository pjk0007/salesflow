# Sales SaaS 프로젝트

## 개요
범용 멀티테넌트 영업(세일즈) 관리 SaaS.
weldy 프로젝트를 참고하되 위들리 종속 코드 없이 새로 구축.

## 기술 스택
- Next.js 16 (Pages Router) + TypeScript
- PostgreSQL + Drizzle ORM
- Tailwind CSS v4 + shadcn/ui
- SWR, Sonner, Lucide Icons
- NHN Cloud Alimtalk, Nodemailer

## 실행 방법
```bash
pnpm install
pnpm dev          # 개발 서버 (Turbopack)
pnpm build        # 프로덕션 빌드
pnpm db:push      # DB 스키마 반영
pnpm db:seed      # 시드 데이터 생성
```
